# Topicverse

A unique, premium-designed interactive mini web application that explores fascinating topics using modern aesthetic web design, built with React, Vite, and tailwind-like utility structures (implemented in raw CSS for custom design). 

This project demonstrates advanced glassmorphism, dynamic gradients, smooth micro-interactions, and modal experiences.

## ✨ Features
- **Premium Aesthetics**: Glassmorphism and gradient layers over a dark cosmic theme.
- **Interactive UI**: Fully functional and animated topic exploration cards using `framer-motion`.
- **Informative Content**: Learn about Quantum Physics, Deep Sea biology, Astrophysics, and Stoic philosophy.
- **Responsive Layout**: Designed to look perfect on both mobile and desktop screens.
- **Micro-Animations**: Uses `lucide-react` for iconography along with smooth hover states and state transitions.

## 🚀 Tech Stack
- Frontend Framework: React (via Vite)
- Animations: Framer Motion
- Icons: Lucide-React
- Styling: Plain CSS with advanced features (variables, CSS Grid/Flexbox, blur filters)

## 📦 Installation & Setup

To run this project locally, simply follow these steps:

1. **Clone the repository** (if you have uploaded this to GitHub):
   ```bash
   git clone <repository_url>
   ```

2. **Navigate to the project directory**:
   ```bash
   cd "mini project 2"
   ```

3. **Install the dependencies**:
   ```bash
   npm install
   ```
   *(This project uses `lucide-react` and `framer-motion`)*

4. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open your browser and navigate to the localhost URL provided in the terminal (usually `http://localhost:5173/`).

## 🛠 Project Structure
- `src/App.jsx` - Contains the main React component logic, state management, and animated components.
- `src/index.css` - Holds all the premium styling rules like glassmorphism and gradient texts.
- `src/App.css` - Cleaned up to establish custom stylings smoothly.
